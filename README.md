PHP tests & code analysis
=========================

Intention
----------
Basic & advanced practice of :
- TDD (Test Driven Development) - `PHPUnit`
- BDD (Behavior Driven Development) - `Behat`
- Code analysis - `PHPStan` & `PHP CS Fixer`.

Dev Env
--------
- `Symfony 6.1` (traditional web app skeleton)
- `PHP 8.1`
- `PHPUnit 9.5`
- `PHPStan 1.8`
- `Behat 3.11`
- `PHP CS Fixer 3.11`
